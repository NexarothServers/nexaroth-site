# 🌐 Website Complet - Plateforme Vote & Boutique

Un site web complet avec inscription, vote, boutique et panier d'achat.

## ✨ Fonctionnalités

✅ **Inscription et Connexion** sécurisée
✅ **Système de Vote** en temps réel
✅ **Boutique** avec produits
✅ **Panier d'achat** et commandes
✅ **Profil utilisateur** et historique
✅ **Base de données SQLite**
✅ **Design moderne et responsive**

## 🚀 Installation

### Prérequis
- Node.js (v14+)
- npm ou yarn

### Étapes

1. **Cloner le projet**
```bash
git clone https://github.com/NexarothServers/nexaroth-site.git
cd nexaroth-site
```

2. **Installer les dépendances**
```bash
npm install
```

3. **Lancer le serveur**
```bash
npm start
```

4. **Accéder au site**
Ouvrez votre navigateur et allez à `http://localhost:3000`

## 📋 Pages disponibles

- 🏠 **Accueil** - Page principale
- 📝 **Inscription** - Créer un compte
- 🔐 **Connexion** - Se connecter
- 🗳️ **Vote** - Participer aux sondages
- 🛍️ **Boutique** - Voir les produits
- 🛒 **Panier** - Gérer les achats
- 👤 **Profil** - Voir son compte et historique

## 🛠️ Technologies utilisées

- **Backend**: Node.js + Express
- **Base de données**: SQLite3
- **Authentication**: JWT + bcryptjs
- **Frontend**: HTML5 + CSS3 + JavaScript

## 📦 Dépendances principales

- express
- sqlite3
- bcryptjs
- jsonwebtoken
- cors

## 🔐 Sécurité

Les mots de passe sont chiffrés avec bcryptjs et les sessions utilisateur sont gérées avec JWT.
